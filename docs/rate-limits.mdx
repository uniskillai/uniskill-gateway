## Subscription Tiers

Different subscription plans have the following rate limits:

<!-- AUTO_GEN:TIER_TABLE_START -->
| Tier | Rate Limit (RPM) | Features |
| :--- | :--- | :--- |
| **Free** | 30 RPM | Entry level access |
| **Starter** | 60 RPM | Basic API access |
| **Pro** | 300 RPM | Priority support, audit logs |
| **Scale** | 1000 RPM | Multi-key management, dedicated bandwidth |
<!-- AUTO_GEN:TIER_TABLE_END -->

*RPM = Requests Per Minute.*
